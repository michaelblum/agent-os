# Cog Radial Asset

Sigil's context-menu radial item loads this native Three.js glTF asset from:

```text
apps/sigil/renderer/assets/models/cog/scene.gltf
```

Source model:

- Title: Cog
- Author: Jiri Kuba
- URL: https://sketchfab.com/3d-models/cog-158a1e27214841589dce6d7361f1a422
- License: CC-BY-4.0

The downloaded Sketchfab glTF archive contents are stored in this directory so
`scene.gltf` and `scene.bin` are served by the Sigil content root.
