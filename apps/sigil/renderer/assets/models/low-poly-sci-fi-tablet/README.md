# Low Poly Sci-Fi Tablet Radial Asset

Sigil's Codex terminal radial item loads this native Three.js glTF asset from:

```text
apps/sigil/renderer/assets/models/low-poly-sci-fi-tablet/scene.gltf
```

Source model:

- Title: Low Poly Sci-Fi Tablet
- Author: Snooze
- URL: https://sketchfab.com/3d-models/low-poly-sci-fi-tablet-ee1fde7ec1514fd5a61790809ebd46a6
- License: CC-BY-4.0

The downloaded Sketchfab glTF archive contents are stored in this directory so
`scene.gltf`, `scene.bin`, and textures are served by the Sigil content root.
