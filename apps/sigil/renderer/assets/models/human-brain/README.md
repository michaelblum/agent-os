# Human Brain Radial Asset

Sigil's wiki radial item loads this translucent brain model from:

```text
apps/sigil/renderer/assets/models/human-brain/scene.gltf
```

Source model:

- Title: Human Brain
- Author: Versal
- URL: https://sketchfab.com/3d-models/human-brain-49bcdf19c1904c76a456b31838b0d7ac
- License: CC-BY-4.0

Runtime files:

```text
scene.gltf
scene.bin
```

The renderer replaces source materials with a translucent material so the wiki
tree network can remain visible inside the model.
