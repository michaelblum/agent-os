# Employer Brand Live Evidence Visibility-Adjusted Capture Plan V0

This artifact promotes the four Operator-filled repaired-capture visibility
repair decisions into deterministic capture-plan slots. It is a planning and
provenance artifact only.

The plan preserves repaired slot identity, target/work-unit identity, company,
source category, approved URL, natural-language target, KILOS relevance,
evidence goal, expected clip count, repaired locator, prior failure metadata,
Operator notes, and the approved scroll/wait/viewport precondition. It also
preserves the LinkedIn source-unavailable slot and the remaining non-executable
context entries.

The artifact must keep accepted capture count and actual asset counts at zero,
keep all planned output paths null, and keep `full_page_grab=false`. Producers
must not open URLs, run capture, resolve locators, create screenshots/clips/text
extracts, bypass access controls, render reports, export documents, or broaden
the target set.
