# Employer Brand Repaired Capture Runtime Diagnostics v0

This schema defines a read-only diagnostics artifact for repaired live evidence
capture attempts that fail before useful capture evidence is produced. It
records the capture command surface, timeout/error metadata, per-slot retry or
review eligibility, non-executable context, and invariants that classify the
failure as a runtime invocation issue, locator failure, or content/access
blocker.

The artifact does not authorize browsing, locator resolution, code generation,
new selectors, screenshots, clips, text extracts, report rendering, exports,
workflow execution, or access-control bypasses.
