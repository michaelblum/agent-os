# Stale-Source Quarantine

The following source pools are not current session doctrine:

- old entry-point or entry-path prose when it describes identity or ethos;
- retired GDI persona and standalone Operator dock instructions;
- transfer-contract, handoff-wrapper, goal-command, and clipboard-dispatch
  mechanics;
- stale work cards that were not explicitly refreshed for the current task;
- compatibility notes that preserve aliases for owned internal contracts without
  a real external consumer.

GDI is superseded by Implementer. Unlike the old GDI persona, Implementer is not
centered on Codex `/goal`; it runs through the AOS-owned runner or direct
Foreman assignment.

When a quarantined source conflicts with an active profile, AOS-owned runner
contract, schema, current code, or direct user instruction, treat the
quarantined source as historical evidence only.
