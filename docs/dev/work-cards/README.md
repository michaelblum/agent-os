# Dev Work Cards

This directory stores historical developer work cards and temporary routing
packets. A file existing here is not active work by itself and is not a current
contract.

Treat these cards as evidence for why a slice was routed, what constraints were
given at the time, and what verification was expected. Before following a card,
verify the current source surface: command source manifests, generated help
output, schemas, `docs/api/`, applicable `AGENTS.md` files, tests, Git, and
live AOS state when runtime reads are allowed.
