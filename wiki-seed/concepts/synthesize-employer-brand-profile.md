# Synthesize Employer Brand Profile

This process produces a reusable [Employer Brand Profile](../entities/employer-brand-profile.md) for one company.

## Inputs

- Normalized evidence bundle
- Agreed message dimensions
- Scope and audience context

## Outputs

- Profile summary
- Dimension-by-dimension assessment
- Proof points and evidence references
- Open unknowns

## Recommended Steps

1. Restate scope and evidence coverage.
2. Identify the strongest employer promise.
3. Map direct evidence to the message dimensions.
4. Describe brand voice and proof structure.
5. Record contradictions, gaps, and unresolved questions.
6. Save the result in a reusable profile format.

## Rules

- Stay company-specific. Do not compare unless the comparison process is the task.
- Name what is evidence, what is inference, and what is still unknown.
- Use concise synthesis language; the citations carry the proof burden.
