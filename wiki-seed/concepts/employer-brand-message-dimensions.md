# Employer Brand Message Dimensions

This workflow set uses a normalized dimension model so profiles and comparisons can be composed without binding the entire system to one proprietary vocabulary.

## Default Families

The default five families are:

- `Kinship` - belonging, culture, inclusion, team experience
- `Impact` - purpose, mission, influence, innovation, scale
- `Lifestyle` - flexibility, benefits, environment, work-life conditions
- `Opportunity` - growth, learning, mobility, stretch, advancement
- `Status` - reputation, recognition, prestige, heritage, leadership

These line up well with the current KILOS-based source material, but this page is the canonical working layer for the repo wiki.

## Orthogonal Lenses

Analyze each family through these additional lenses:

- Audience: who the message is for
- Channel: where the message appears
- Proof: what evidence supports it
- Tone: how the message sounds
- Region: whether the message is global or local
- Call to action: what response the message tries to drive

## Mapping Rules

- One piece of evidence can map to multiple dimensions.
- Presence is not the same as strength.
- Local or role-specific messaging should not be mistaken for global positioning.
- Channels matter: a message that appears only on social is different from one anchored on the careers site.
