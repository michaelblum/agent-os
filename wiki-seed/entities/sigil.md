# Sigil

Sigil is an external first-party product built on AOS. Active product source,
documentation, packaging, and tests live in
[`Ch-osctrl/sigil`](https://github.com/Ch-osctrl/sigil).

## Embedded Fixture

`apps/sigil/` is a frozen legacy compatibility fixture. It is not discoverable,
packaged, launched, or accepted as product behavior by AOS. Git history retains
the former embedded architecture; active work must follow the external product
repository.
